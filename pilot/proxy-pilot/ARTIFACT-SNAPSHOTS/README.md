# Artifact Snapshots

Save dated copies of the shared coordination doc here:
- End of Phase-2 week 1 (pilot ~day 14)
- End of Phase-2 week 2 (pilot ~day 21)
- Immediately after the pressure event (Template D window)

Naming: artifact-YYYYMMDD.md (or .csv / export). These are RAW evidence (PR-2).
Versioned snapshots let you show the artifact's state at each decision point and
catch decay early (snapshot at first sign of a lapse, not only scheduled points).
