  Raw transcript (here) -> Evidence Intake -> Field Log -> Assumption Tracker -> Ledger -> Journal

RULES (per PR-1 + PR-2):
1. Every practitioner reply received (LinkedIn / email / screenshot / paste / export / chat
   log) is written HERE as a raw file BEFORE any summary, intake, or assumption note.
   Naming: <SOURCE>-<DATE>.md (e.g. Mones-MSF-2026-07-17.md, Hisem-WhiteHelmets-2026-07-17.md).
2. The raw file contains the VERBATIM wording — no paraphrasing, no editing, no summarising.
   Add metadata (channel, date, questions asked) but keep the practitioner's words intact.
3. Evidence Intake, Field Log, and Assumption Tracker MUST cite the raw-transcripts/ file as
   the primary source. A summary in those layers points BACK to this file; it never replaces it.
4. No assumption may cite a summary when a primary raw transcript exists (PR-2).
