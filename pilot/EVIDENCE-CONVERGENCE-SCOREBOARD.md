# Evidence Convergence Scoreboard

Tracks, per assumption, how many **independent sources** point the same way. The
project's strongest signal is **convergence** — three independent sources agreeing
beats any single report or interview. Update this board after every evidence event
(new source retrieved, new interview logged, new reply).

## Column definitions

- **Public** — structural / positional public evidence retrieved (datasets, indexes,
  institutional sources). Count = distinct [RETRIEVED] public sources in
  PUBLIC-EVIDENCE-REVIEW.md.
- **First-Hand** — published **primary-field accounts**: after-action reviews,
  practitioner testimonies, field bulletins that report a real event in someone's own
  words. Distinct from structural datasets.
- **Interview** — direct conversations / replies logged: full interviews
  (assumption-log/*.md) + partial replies that touch the assumption
  (CONTACT-STATUS.md).
- **Convergence Status** — Weak / Emerging Convergence / Converged / Contradicted /
  Untested, judged by independent-source agreement, not raw counts.
