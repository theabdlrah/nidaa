# Nidaa — G1 / G2 Closure Criteria (Day 10)

Type: EVIDENCE AUDIT deliverable (Day 10, Task 3). Auditor role.
Purpose: specify the EXACT practitioner statement sufficient to close G1 (informal SUCCESS)
and G2 (informal COLLAPSE). The only valid instrument is the six-question gate
(docs/Nidaa-A6b-Six-Questions.md). Literature cannot close G1/G2.

==================================================================
G1 — INFORMAL SUCCESS CASE (closure criteria)
==================================================================

G1 asks: did a grassroots/informal coordinator maintain structured operational information
that improved coordination outcomes, and WHY did it hold?

SUFFICIENT PRACTITIONER STATEMENT (verbatim six-question-gate response from a REAL
coordinator in the actual Nidaa population — connectivity-stressed, protracted crisis,
grassroots/informal, NOT a semi-formal org with a standing mandate):

  "We [a named/designated owner or mandate existed for the coordination info — e.g. 'our
   info focal point / committee secretary / a designated person owned it']. We kept a
   [spreadsheet / register / field report / shared doc] actively updated with
   [beneficiary lists / needs / incidents / distributions / status]. During [a concrete
   pressure event: a connectivity blackout / surge of incidents / insecurity / burnout],
   the structured record [survived / kept getting updated] because [owner + low-burden
   capture + fallback path — name the mechanism]. It [improved matching / prevented
   duplication / cut wrong-info / let us act when chat alone would have failed]."

MINIMUM CLASSIFICATION REQUIREMENTS for a G1 closure:
  (i)   a designated owner/mandate is named (this is what separates G1 from a chat habit),
  (ii)  structured info was actively maintained (not just WhatsApp),
  (iii) it held through a SPECIFIC pressure event, with the WHY stated,
  (iv)  the coordinator is from the loose-informal population Nidaa targets (Gaza/Syria
        mutual-aid, camp/neighborhood committee, community initiative) — NOT semi-formal
        White Helmets or formal MSF (those already provide semi-formal support).

NOTE ON CURRENT STATE: G1 is CLOSED ONLY AT THE SEMI-FORMAL TIER (White Helmets field/monthly
reports per ACAPS; Mones/MSF formal-end owner/mandate). This satisfies "structure CAN hold
with owner/mandate" but does NOT close G1 for the actual Nidaa population. G1(tight) =
still OPEN until a loose-informal coordinator supplies the statement above.

G1(tight) closure also requires the statement to be recorded verbatim in the Evidence Ledger
tagged G1, with Confidence = high (concrete own-case), per the ledger rubric.

==================================================================
G2 — INFORMAL COLLAPSE CASE (closure criteria)
==================================================================

G2 asks: did a grassroots/informal coordinator's structured/attempted coordination collapse
or create confusion/duplication/misinformation, and WHY did it break?

SUFFICIENT PRACTITIONER STATEMENT (verbatim six-question-gate response from a REAL
coordinator):

  "We [started / planned] a [spreadsheet / board / register / shared doc] to coordinate
   [needs/offers/incidents]. Under [a concrete pressure event: surge / connectivity loss /
   insecurity / burnout] it [collapsed / died / was abandoned] — [e.g. 'the document died
   after three days'; 'everyone fell back to calls and chat']. It broke first because
   [no designated owner / the updating burden was too high / connectivity cut it / the
   coordinator burned out / nobody owned verification]. The consequence was
   [duplicated effort / conflicting info / a need went unmet / wrong info spread]."

MINIMUM CLASSIFICATION REQUIREMENTS for a G2 closure:
  (i)   structured/attempted coordination existed or was intended (not "we never tried"),
  (ii)  a concrete pressure event is named,
  (iii) the failure mode + WHY is stated (owner absence / burden / connectivity / burnout),
  (iv)  ideally a consequence (duplication / misinformation / missed need) is described,
  (v)  recorded verbatim in the Evidence Ledger tagged G2, Confidence = high.

WHY LITERATURE CANNOT CLOSE G2: Solnit's G2 material is ELITE-driven (rumors from info-
withholding elites), NOT informal-networks-failing-from-within. A genuine G2 must come from
a practitioner describing their own coordination breaking. The book under-addresses A6b's G2.

CURRENT STATE: G2 is ENTIRELY MISSING. No practitioner has described an informal-collapse
event. This is the single largest evidence gap. Highest-value future reply = a coordinator
saying "we tried to structure it and it died" with the WHY.

CLOSURE GATE (both G1 and G2)
==================================================================

Neither G1 nor G2 is closed by:
  - a literature source (Solnit, ACAPS, WH reports),
  - a "liked"/read receipt/connected status,
  - an opinion ("I think informal networks help"),
  - a semi-formal org standing as the loose-informal case.

Both are closed ONLY by a first-hand coordinator story answering the six questions, logged
verbatim in the Evidence Ledger with Confidence = high, from the actual Nidaa population,
stating the owner/mandate condition and the pressure-event outcome with the WHY.

Validation of A6b requires convergence across INDEPENDENT sources (constitution rule), not a
single G1 or G2. One story is evidence; convergence is validation.
